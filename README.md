
## Database 
To have a working Database please start a new Docker container and run the sql(create.sql) script

```docker create -v /var/lib/postgresql/data --name PostgresData alpine```

```docker run -p 5432:5432 --name postgresql -e POSTGRES_PASSWORD=postgres -d --volumes-from PostgresData postgres```

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
